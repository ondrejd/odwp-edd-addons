== Description ==

Úpravy pro WordPress plugin Easy Digital Downloads.

== Installation ==

1. Stáhněte poslední verzi pluginu z repozitáře se zdrojovými kódy
2. Rozbalte archív do adresáře s pluginy vaší instalace aplikace WordPress
3. Aktivujte plugin v administraci aplikace Wordpress
